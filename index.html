Gemini in Application

Built an Android chatbot app using Jetpack Compose for a clean UI and interacted with Google AI's Gemini model for natural language conversations. Face Detection

I built a face detection application using a machine learning model. The app can pinpoint faces in an image and estimate various facial features. It outputs the probability of a smile and the percentage openness of both eyes. This project allowed me to explore the applications of machine learning for real-time image analysis and facial recognition.

Shooter (game)

This is a basic Unreal Engine shooter project I built to explore core game development functionalities. Players control their character's movement (WASD) and engage enemies with AI-driven chasing behavior. The project implements shooting mechanics along with sound effects for immersion (gunfire). Basic animations for movement and enemy attacks further enhance the gameplay experience. This project served as a valuable learning experience in Unreal Engine's capabilities for character control, enemy AI, and core gameplay mechanics.

Student Dropout Analysis System

Our project is a machine learning-based application designed to predict and prevent student dropout in educational institutions. By analyzing student data such as demographics, academic performance, and socioeconomic status, our system provides early warnings and intervention strategies to support at-risk students. Our goal is to empower educators with actionable insights to foster student success and retention.

Contact Manager App

I developed a secure contact manager app with user registration and login for enhanced data protection. Users can easily add new contacts including names, phone numbers, and email addresses. Information for any contact can be retrieved by simply searching their name, making it a user-friendly and reliable tool for managing personal and professional connections.
